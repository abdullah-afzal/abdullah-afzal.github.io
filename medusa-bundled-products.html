medusa-bundled-products

A Medusa plugin that adds bundled product functionality — kits, starter packs, and promotional combos.

Repository & Documentation

What is it?

medusa-bundled-products allows you to create and manage bundled products — multiple products sold as a single offering. Use it for starter kits, promotional combos, or curated packs.

Features

  • Create bundled products with linked products/variants
  • Set custom bundle pricing
  • Seamless integration with Medusa product and variant models
  • Cart & order handling for bundles
  • Storefront-friendly data shape for easy UI display

Compatibility

This plugin is compatible with @medusajs/medusa >= 2.4.0.

Installation

  1. Install the package with yarn add medusa-bundled-products or npm i medusa-bundled-products
  2. Add to medusa-config.js: 
                            module.exports = defineConfig({
                            // ...
                            plugins: [
                            {
                                resolve: "medusa-bundled-products",
                                options: {
                                    // optional plugin options
                                },
                            },
                            ],
                        })

Development

Run migrations:

npx medusa db:migrate

API Endpoints

POST /admin/bundled-products — Create a bundled product (admin)

GET /admin/bundled-products — List bundled products (admin)

GET /store/bundled-products/:id — Get bundle details (store)

POST /store/cart/:id/line-item-bundles — Add bundle to cart

DELETE /store/cart/:id/line-item-bundles/:bundle_id — Remove bundle from cart

Cart Behavior

Bundles are treated as a single logical unit in the cart. When a bundle is added:

  • All component items are added together (with quantities)
  • Bundle pricing is applied (price override or computed sum)
  • Inventory checks are performed for each component
  • Removing the bundle removes all its component items from the cart
